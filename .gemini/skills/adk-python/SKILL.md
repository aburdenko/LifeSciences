# Agent Development Kit (ADK) Documentation

## Overview

This skill provides comprehensive documentation and guidelines for the Agent Development Kit (ADK), particularly the Python implementation. ADK is a flexible and modular framework for developing, orchestrating, and deploying AI agents.

## Core Reference

The complete documentation for ADK is available in the bundled references file:

- **Complete ADK Documentation**: Read `references/adk-docs.md` to learn about:
  - Agent types (`LlmAgent`, `WorkflowAgent`, Custom Agents)
  - Integrating models (Gemini, Anthropic, LiteLLM, local models)
  - Tool creation and usage
  - Multi-agent orchestration and workflows
  - Session and state management

Always use search and file reading tools on `references/adk-docs.md` to search for code examples, API details, or specific features when assisting users with ADK-related tasks.

To utilize this skill effectively:
